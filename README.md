# Family Wealth Hub

A full Next.js starter website for a private family savings and investing portal.

## Included pages

- Dashboard
- Members
- Meetings
- Motivation
- Goals
- Resources
- Admin
- Login

## Run locally

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```
